⚙ Google 시트 연동 설정

    
Apps Script 배포 URL을 입력하면 2대 PC에서 데이터가 자동 공유됩니다.

    

      
배포 URL

      
    

    

      📋 설정 순서

      1. Google 시트 열기 → 확장 프로그램 → Apps Script

      2. MeClinic_AppScript.js 코드 붙여넣기 후 저장

      3. 배포 → 새 배포 → 웹 앱 → 액세스: 모든 사용자

      4. 생성된 URL 위에 붙여넣기
    

    

      🖥 원장님 대시보드 URL
